Häufig gestellte Fragen (FAQ)

Die App zeigt keine Daten an – was tun?

Stelle sicher, dass du eine aktive Internetverbindung hast. Die App lädt und aktualisiert die Inhalte beim ersten Start online.

Wann wird die Running Order veröffentlicht?

Die Running Order erscheint in der App, sobald sie vom Festival veröffentlicht wurde.

Wie oft werden Daten aktualisiert?

Änderungen am Line-up oder Shuttle-Fahrplan werden regelmäßig über App-Updates oder automatisch eingespielt.

Ich habe einen Bug gefunden – wohin damit?

Fehler kannst du direkt über GitHub Issues melden oder per E-Mail an uns senden.
